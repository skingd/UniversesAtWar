# This is used for item name correction

**CL**: Clan
**IS**: Inner Sphere
**MGA** Machine Gun Array

`CL` Replace with `Clan`
`IS` Replace with `Inner Sphere`
`3 Slave Unit`: Replace with `C3 Slave Unit`
`MGA` Replace with `Machine Gun Array`
`ISAC10` Replace with `Inner Sphere AC/10`
`ISAC5` Replace with `Inner Sphere AC/5`
`ISAC20` Replace with `Inner Sphere AC/20`
`ISAC2` Replace with `Inner Sphere AC/2`
`ISCASE` Replace with `CASE`
`CLCASE` Replace with `CASE`


`Long Tom Cannon` This is the `Long Tom Cannon` Weapon, not a Special Rule
